mirrorimage Module

Description

Simple module that allows image horizontal flipping (Mirror). Useful when taking pictures with the frontal camera.

Installation

http://wiki.appcelerator.org/display/tis/Using+Titanium+Modules

Changelog

Accessing the mirrorimage Module

To access this module from JavaScript, you would do the following:

var mirrorimage = require("com.inzori.mirrorimage");

The mirrorimage variable is a reference to the Module object.

Reference

This module exposes only one method:

mirrorImage.imageMirror(imageBlob)

Usage

// convert imageView to image blob var imageBlob = iv.toImage();

// the parameter must be a blob, returns a blob var newImageBlob = mirrorImage.imageMirror(imageBlob);
